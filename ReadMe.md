# A Simple Compiler
###### by Russell Ratcliffe

## How to run program on Linux
1. open terminal 
2. `git clone https://github.com/russeii/compilers`
3. `cd compilers`
4. `mkdir make`
5. `cd make`
6. `cmake ..`
7. `make`
8. `./bool`


## Implementation
My language supports two different types. Integers and Boolean values.
